# FlagOS SIG

## 小组简介

**FlagOS SIG** 致力于负责 FlagOS 开源软件包在 deepin 上的移植、发布和维护。发布基于 deepin 的 FlagOS 版本，并进行 FlagOS 软件包打包、BUG 修复等工作。

FlagOS（众智 FlagOS）是由北京智源研究院牵头、多家国内外机构共建的开源统一 AI 系统软件栈，定位为 “大模型时代的跨芯片操作系统”，解决 AI 硬件生态碎片化、模型跨芯片迁移成本高的痛点。

## 活动范围与目标

活动范围：

[GitHub 上的小组仓库](https://github.com/deepin-community/sig-flagos)

[FlagOS GitHub](https://github.com/flagos-ai/)

目标：

1. **完成 FlagOS 在 deepin 上的适配和移植**

   确保 FlagOS 软件栈能够在 deepin 操作系统上稳定运行，并进行必要的功能优化。
2. **负责 FlagOS 在 deepin 的软件包更新和维护**

   持续跟进 FlagOS 上游版本，进行软件包打包、更新和 BUG 修复等工作。
3. **构建基于 deepin 和 FlagOS 组合的开源演示方案**

   提供可复现的 FlagOS on deepin 演示方案，方便开发者和研究者基于 deepin 使用 FlagOS 进行 AI 模型的训练和推理。

## 小组章程

### 加入标准：

认同本 SIG 目标，对 FlagOS、AI 系统软件栈感兴趣，愿意参与 FlagOS 在 deepin 上的移植与维护工作。
