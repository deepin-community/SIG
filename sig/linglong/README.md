# linglong SIG

## 小组简介

本 SIG 主要进行“玲珑”容器化包管理工具相关的讨论。

## 活动范围与目标

活动方式：于 [Matrix](https://matrix.to/#/#linglong:matrix.org) 服务器以及相关项目的 Issue 区中进行相关讨论，讨论内容包括但不限于 bug 上报以及相关技术讨论。

主要目标为推进与进行“玲珑”相关工具的开发与维护。

## 小组章程

- 友好地进行“玲珑”相关工具/技术方面的讨论
