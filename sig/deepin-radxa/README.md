# deepin-radxa SIG

## 小组简介

**deepin-radxa SIG 小组** 致力于将 deepin 操作系统成功移植到 Radxa 嵌入式平台，并持续优化系统的性能与兼容性。

## 活动范围与目标

活动范围：

[GitHub](https://github.com/radxa/)

[Radxa Forum](http://forum.radxa.com)

目标：

1. **完成 deepin 在 Radxa 平台的完整移植与适配**

   确保 deepin 操作系统能够在 Radxa 的多种嵌入式设备上稳定运行，并进行必要的功能优化。
2. **提升系统性能与硬件兼容性**

   针对 Radxa 平台的硬件特性，优化 deepin 系统的启动速度、资源占用、图形渲染及驱动支持，确保用户体验良好。
3. **构建和维护适合嵌入式场景的 deepin 镜像与工具链**

   提供定制化的 deepin 嵌入式镜像，简化开发者的部署流程，并维护相关开发工具与 SDK 的可用性。
4. **推动开源社区协作与知识共享**

   通过社区活动、技术分享和文档建设，加强 deepin 与 Radxa 生态之间的联系，促进更多开发者参与进来。

## 小组章程

对瑞莎的单板机及其周边产品感兴趣，想移植 deepin 系统到瑞莎单板机及其周边产品。
